Self Hosting Mudah dan Gratis

Dipublish: 17 Nov 2021

Waktu baca: 2 menit

Saat kita sedang dalam proses developing website tentunya kita akan pake yang namanya local server, local server merupakan tool bawaan framework atau dari 3rd party library yang bakal membuatkan server palsu di komputer untuk menjalankan website kita.

Local server walau sangat berguna namun masih terdapat limitasi seperti cuman bisa diakses di komputer itu aja ataupun maximal hanya bisa diakses dalam satu local network. Apabila kita butuh ngetes suatu fitur di website kita apa sudah layak diakses dari internet maka mau tidak mau website harus diupload dulu ke development server atau hosting, tentunya ribet sekali dan overkill kalau harus nyiapin server hanya untuk tujuan itu.

Nah sekarang sudah ada solusi agar localhost kita dapat diakses via internet menggunakan localtunnel. Sebelum menggunakan aplikasi ini jangan lupa install nodejs di OS kesayangan anda jika sudah terinstall cek dengan perintah

node -v

kemudian bikin folder baru lalu jalankan command

npm init -y

pindah ke folder yg baru dibuat lalu install localtunnel

npm i --save-dev localtunnel

jika sudah, silahkan jalankan local server yang ingin di tunnel perhatikan url beserta portnya contohnya: http://localhost:8080/ kemudian jalankan localtunnel dengan argument port yang sama dengan yg ada di local server

npx lt --port 8080

hasil dari perintah diatas akan menampilkan url baru contohnya: https://flkajsfljas.loca.lt. Selamat website anda udah terhubung ke internet, nah dari sini terserah anda apakah url ini mau dishare ke client buat demo atau ke temen-temen buat pamer direview dan dites.

Kelebihan

  1. Deployment sangat mudah daripada cara biasa
  2. Gak repot mikirin infrastruktur seperti server,hostingan dll karena yang jadi server adalah komputer kita sendiri (self hosted)
  3. Sangat cocok buat testing atau quick demo karena website sangat mudah untuk dionlinekan dan diofflinekan
  4. Open Source

Kekurangan

  1. Url yang didapat random dan bersifat sementara
  2. Karena self hosted, jika local server dimatiin atau komputernya di matiin maka web tidak akan bisa diakses lagi dari internet
  3. Ada resiko keamanan karena terbukanya akses dari internet ke komputer kita. So be careful
  4. Terkadang server localtunnel bisa mengalami down jadi kalo bermasalah ya sabar aja atau cari alternatif lain disini

Mohon maaf, resolusi layar anda terlalu rendah.
