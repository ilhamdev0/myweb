Trik Menggabung Isi File Dengan Cepat

Dipublish: 08 Jan 2022

Waktu baca: 3 menit

Bayangkan kamu punya 2 buah file yang isinya tulisan dan kamu pingin isi dari file tadi digabung jadi satu, apakah yang akan kamu lakuin?

a.txt

b.txt

Old Way

mungkin kamu akan melakukan step berikut ini:

  1. buka salah satu file di editor
  2. copy seluruh isi file
  3. paste ke dalam file yg lain
  4. selesai

atau seperti ini

  1. buat file baru (untuk menampung isi dari kedua file)
  2. buka salah satu file lalu copy seluruh isinya
  3. paste ke file baru
  4. buka file yg lain lalu copy seluruh isinya
  5. paste ke file baru
  6. selesai

Terlihat sangat mudah dilakukan bukan, sekarang bayangkan kita punya 10 file yang ingin digabung

lorem.txt       sit.txt                       adipisicing.txt       do.txt

ipsum.txt      amet.txt                  elit.txt

dolor.txt        consectetur.txt       sed.txt

kita masih bisa menggunakan cara diatas, no problem what so ever.. namun sekarang bayangin misal ada 100 file yang harus digabung disini mulai nampak kelemahan kita sebagai manusia yaitu:

  • Rugi waktu

    ngerjain 100 file butuh berapa puluh menit ya?

  • Rugi effort

    • yakin dari 100 file g ada yg kelewat sama sekali?
    • yakin selama pengerjaan gak ngepaste 2 kali tanpa sengaja atau lupa di paste?
    • gimana cara ngecek kalo pengerjaannya udah bener?
    • kalo ada salah gimana cara perbaikinnya?

kalo anda tidak punya kelemahan diatas selamat anda adalah seorang superman hehe.. karena penulis adalah seorang manusia biasa yang penuh kelemahan maka pakenya cara seperti berikut

New Way

let say misalnya ada 15 file yang ingin digabung

lorem.txt       consectetur.txt     eiusmod.txt

ipsum.txt      adipisicing.txt      tempor.txt

dolor.txt        elit.txt                    incididunt.txt

sit.txt             sed.txt                   ut.txt

amet.txt        do.txt                     labore.txt

Langkah pertama taruh seluruh file dalam satu folder yang sama (nama foldernya bebas)

> folder_tes

        lorem.txt       consectetur.txt     eiusmod.txt

        ipsum.txt      adipisicing.txt      tempor.txt

        dolor.txt        elit.txt                    incididunt.txt

        sit.txt             sed.txt                   ut.txt

        amet.txt        do.txt                     labore.txt

Langkah kedua buka terminal lalu arahkan ke dalam folder tadi

cd folder_tes

Langkah ketiga ketik command berikut

cat * > hasil.txt

command diatas akan menggabungkan seluruh file lalu outputnya akan disimpan ke dalam file baru bernama "hasil.txt" (nama file bisa diganti sesuka hati)

sangat simple sekali bukan, dengan cara diatas walau terdapat 100, 1.000, bahkan 100.000 file sekalipun bisa digabung dengan sangat cepat dan udah pasti diproses seluruhnya dengan benar tanpa kelewat satupun.

satu lagi kelebihan cara baru ini adalah kita bisa menentukan urutan penggabungan file dengan jalan merename file tadi

> folder_tes

        02lorem.txt       01consectetur.txt     05eiusmod.txt

        10ipsum.txt      11adipisicing.txt      03tempor.txt

        04dolor.txt        08elit.txt                    09incididunt.txt

        06sit.txt             07sed.txt                   12ut.txt

        13amet.txt        14do.txt                     15labore.txt

cat * > hasil.txt

file akan diproses sesuai urutan nomor, pada file "hasil.txt" isi file dari nomor awalan 01 akan berada di posisi paling atas diikuti nomor selanjutnya sampai akhirnya isi file nomor awalan 15 akan berada di posisi paling bawah.

Real World Example

Cara diatas pernah digunakan oleh penulis saat menangani project scraping data dengan jumlah lebih dari 300 file berformat .csv, masing-masing file memiliki isi setidaknya ratusan baris menggunakan command diatas seluruh file dapat digabung menjadi satu file .csv tunggal secara sempurna dengan jumlah baris mencapai 270.000 lebih terurut secara rapi dalam sekejap mata

Mohon maaf, resolusi layar anda terlalu rendah.
